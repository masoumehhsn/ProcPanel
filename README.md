# Process Panel (C++ / Qt / QML)

A lightweight Process Monitoring Panel for Linux built with C++, Qt, and QML.
It provides real-time system process visualization and monitoring with a modern UI.

![alt text](https://github.com/masoumehhsn/ProcPanel/blob/main/res/screenshot.png)

## 🚀Features

### Real-time process monitoring
Displays:
- **Process name**
- **PID**
- **CPU usage**
- **Memory usage**

### Sort processes:
- **Ascending / Descending order**

### Process control support:
- **Suspend process**
- **Resume process**
- **Stop process**

### Live charts for:
- **CPU usage**
- **Memory usage**

- **Responsive UI built with QML**
- **Linux-based system monitoring using /proc**

### 🛠️ Tech Stack
- **C++17**
- **Qt / Qt Quick / QML**
- **Linux /proc filesystem**
- **Model-View architecture**
